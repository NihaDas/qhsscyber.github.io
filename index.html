About

QHSS Cyber is a web development club in Queens High School for the Sciences at York College (QHSS@YC). We create websites and learn the trending web dev technologies together! You don't have to know anything about programming to join—just be willing to learn and dedicate your time to the club. Most of our current members learned the necessary material last year, and we'll help catch you guys up to speed!

Just being in this club is high key useless. You need to put in the time to build your own project portfolio and skillset. This club only helps you with the process. Having a great website project portfolio helps a lot in landing web dev internships and taking initiative to learn something new looks good to colleges!

Right now, we mainly work with HTML and CSS (Bootstrap). However, more advanced members also use Javascript (NodeJS, React, JQuery, etc.).

Meetings are held every Wednesday in room 214, and the advisor is Mr. Mondestin. They are mainly held for status updates or general group planning and brainstorming. Most of the work is done outside of club time, so you only get what you make of this club.
